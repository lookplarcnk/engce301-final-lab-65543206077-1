## react-nodejs-example
Example Project demonstrating how to develop React application with Nodejs

## Example How To Run API on command prompt

  
- `npm install --force`: install npm.
- `npm start`: Start the project.

**Example:**

  
```command Prompt

npm install -- force
npm start

```
